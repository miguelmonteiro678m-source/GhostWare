# GhostWare
README – GhostWare
Project Overview | Visão Geral do Projeto | Resumen del Proyecto

EN: GhostWare is a modular firmware for the LilyGO T-Dongle S3, designed for pentesting, security testing, Wi-Fi and BLE scanning, and USB automation. It includes a web-based control interface and a fully customizable display. All features are intended for educational, authorized testing, and pentesting purposes. Any irresponsible or illegal use of this firmware is not our responsibility.

PT: GhostWare é um firmware modular para o LilyGO T-Dongle S3, projetado para pentesting, testes de segurança, varredura de Wi-Fi e BLE, e automação via USB. Inclui uma interface de controle baseada em web e uma tela totalmente customizável. Todas as funcionalidades são voltadas para fins educacionais, testes autorizados e pentesting. Qualquer uso irresponsável ou ilegal deste firmware não nos responsabilizamos.

Features | Funcionalidades | Funcionalidades
Wi-Fi

EN:

WiFi Scanner — Scans nearby networks and displays SSID, channel, signal strength, and encryption type
Monitor Mode — Captures 802.11 packets and detects handshakes (within hardware limits)
Create AP — Creates a customizable Access Point for testing purposes
Captive Portal (Evil Portal) — Educational captive portal for awareness and credential testing in controlled environments
Deauth Simulation — Simulates client disconnection in authorized test environments
WiFi Beacon Flood (Controlled) — Generates multiple SSIDs for stress testing and analysis

PT:

WiFi Scanner — Escaneia redes próximas e mostra SSID, canal, intensidade do sinal e criptografia
Monitor Mode — Captura pacotes 802.11 e detecta handshakes (dentro das limitações do hardware)
Create AP — Cria um ponto de acesso personalizável para testes
Portal Cativo (Evil Portal) — Portal educacional para conscientização e testes de credenciais em ambientes controlados
Deauth Simulation — Simula desconexão de clientes em ambientes autorizados
WiFi Beacon Flood (Controlado) — Gera múltiplos SSIDs para testes de estresse e análise

ES:

WiFi Scanner — Escanea redes cercanas y muestra SSID, canal, intensidad de señal y cifrado
Monitor Mode — Captura paquetes 802.11 y detecta handshakes (según las limitaciones del hardware)

Portal Cautivo (Evil Portal) — Portal educativo para concienciación y pruebas de credenciales en entornos controlados
Deauth Simulation — Simula la desconexión de clientes en entornos autorizados
WiFi Beacon Flood (Controlado) — Genera múltiples SSIDs para pruebas de estrés y análisis

USB

EN:

BadUSB (HID Emulation) — Simulates a USB keyboard to execute automated scripts and payloads for pentesting

Installation | Instalação | Instalación

EN:

Flash the GhostWare firmware to the LilyGO T-Dongle S3 using ESP-IDF or Arduino.
Connect the device to a PC; the web server starts automatically.
Open the web interface using the assigned local URL.
Upload JPG/GIF images for the device display if desired.

PT:

Grave o firmware GhostWare no LilyGO T-Dongle S3 usando ESP-IDF ou Arduino.
Conecte o dispositivo ao PC; o servidor web inicia automaticamente.
Abra a interface web usando a URL local fornecida.
Faça upload de imagens JPG/GIF para a tela do dispositivo, se desejar.

ES:

Flashea el firmware GhostWare en el LilyGO T-Dongle S3 usando ESP-IDF o Arduino.
Conecta el dispositivo al PC; el servidor web se inicia automáticamente.

Sube imágenes JPG/GIF para la pantalla del dispositivo si lo deseas.
